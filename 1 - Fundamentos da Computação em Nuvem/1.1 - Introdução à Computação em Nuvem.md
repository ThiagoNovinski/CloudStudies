# Módulo 1 - Como dispositivos se comunicam

## Redes de Computadores
- **Dados**: Dados são informações que são traduzidas para um formato eficiente para que sejam transmitidas e processadas nas redes de computadores. 
- **Cabos, fibra óptica, sinal wireless**: Conexões físicas como cabos, fibra óptica e sinais wireless conectam os dispositivos a uma rede de computadores.
- **Adaptadores de redes**: Eles são a interface entre o computador e a rede externa. Todo dispositivo, como um computador, tablet ou servidor que se conecta a uma rede, envia e recebe dados por meio de um adaptador de rede. 
Roteadores, computadore e hubs - Esses dispositivos são responsáveis por direcionar o fluxo de dados e garantir que eles cheguem ao seu destino.
- **Protocolo de rede**: Protocolos de rede, como o Protocolo de Controle de Transmissões/Protocolo da Internet (TCP/IP), são regras que controlam como os dispositivos transmitem dados pela rede.  

## Evolução até chegar em computação em nuvem
- **Servidores em supercomputadores**: No início da Internet, as empresas usavam computadores de alta potência para hospedar páginas da web e armazenar arquivos. Cada computador precisava executar vários websites e o armazenamento do computador continha salvos nele todos os arquivos necessários para os sites.
- **Fazenda de servidores**: As fazendas de servidores foram desenvolvidas, locais onde centenas e depois milhares de servidores se conectavam em rede para compartilhamento das tarefas ou cargas de trabalho envolvidas na entrega de centenas de websites e serviços.
- **Nuvem**: A nuvem é uma evolução dessas fazendas de servidores. Clusters de computadores especializados em armazenamento ou processamento de dados, por exemplo, disponibilizam esses recursos para serem utilizados de forma modular, ou seja, de acordo com a necessidade computacional


# Módulo 2 - Como a computação em nuvem funciona?

## Os Componentes da Nuvem
- **Computadores físicos**: Todos os sistemas em nuvem são compostos por centenas e, muitas vezes, milhares de computadores. Esses computadores executam funções como armazenamento de dados e processamento de informações.
- **Computadores virtuais**: A maioria dos sistemas em nuvem usa computadores virtuais, também conhecidos como máquinas virtuais, para executar várias funções. A VM emula todas as funções do hardware físico. Muitas VMs podem ser executadas em um computador físico e cada VM é executada independentemente de qualquer outra.
- **Dispositivos de rede**: Hardwares e softwares de rede inteligentes e potentes ajudam a assegurar que os sistemas em nuvem consigam mover dados pela Internet de forma rápida e precisa.
- **Software de gerenciamento**: Os administradores usam software de gerenciamento para gerenciar e monitorar a infraestrutura e os serviços em nuvem, o que lhes permite oferecer recursos, monitorar o uso e cumprir políticas de segurança.
- **Software de serviço**: Semelhante a uma interface gráfica com o usuário em um sistema operacional como Windows ou MacOS, os usuários interagem com o software de serviço para acessar o software e os serviços fornecidos pela nuvem.
- **Observação de funcionamento**: Os administradores usam o software de gerenciamento para gerenciar e monitorar os computadores, físicos e virtuais, na nuvem. O usuário interage com o software para acessar os serviços e dados disponíveis nesses computadores. 

## O que é um data center
Um data center  é uma instalação que hospeda sistemas de computador e componentes relacionados, como sistemas de telecomunicações e de armazenamento. Ele foi projetado para fornecer um ambiente controlado e seguro para a operação confiável da infraestrutura em nuvem, que pode incluir servidores, equipamento de rede, sistemas de armazenamento e outros hardwares.

## Composição de um data center
- **Controles ambientais**: Os data centers exigem grandes quantias de eletricidade para alimentar o hardware da nuvem. Eles usam sofisticados sistemas de resfriamento para manter os níveis adequados de temperatura e umidade. 
- **Segurança física**: Os data centers são projetados para serem altamente seguros, com medidas como câmeras de segurança, sistemas de controle de acesso e autenticação biométrica para evitar acesso físico não autorizado.
- **Sistemas de armazenamento**: A maioria dos servidores inclui algum recurso de armazenamento local, como também armazenamento em rede para vários servidores usarem.
- **Conectividade de rede**: Os data centers devem se conectar a redes confiáveis e de alta velocidade para permitir a transferência de dados entre hardware, software e usuários da nuvem.
- **Redundância e recuperação**: Para assegurar máximo tempo de atividade e confiabilidade, os data centers geralmente incluem sistemas redundantes de energia e resfriamento, geradores de backup e múltiplas conexões de rede.

## Pipeline de funcionamento de um data center
- **Envio de dados**: Os dados de usuários e sistemas são enviados pela Internet para um data center que os processa.
- **Segurança**: Os data centers têm muitas medidas de segurança em vigor, como firewalls e controles de acesso de usuário para assegurar que dados maliciosos não atinjam o data center.
- **Software de gerenciamento**: Software gerencia como os dados são processados. O software de gerenciamento garante que os dados sejam direcionados para o local correto.
- **Armazenamento de dados**: Alguns dados são armazenados no data center para serem usados posteriormente. Outros dados são processados pelo software de análise e gerenciamento de banco de dados e enviados de volta aos usuários.
- **Serviços**: O data center processará dados para usuários que configuraram serviços como streaming de vídeo ou gerenciamento de conteúdo. Depois que o serviço processa os dados, eles são enviados de volta aos usuários pela Internet.

# Módulo 3 - Por que as pessoas usam computação em nuvem?

## Principais caracteristicas da computação em nuvem
- **Computação Distribuída**: A computação em nuvem distribui o hardware e as funções de hardware por centenas e, na maioria dos casos, milhares de computadores. Na nuvem, alguns computadores são dedicados a apenas uma função, como armazenamento ou capacidade de processamento. O software coordena essas funções para que as pessoas possam usar diferentes combinações dessas funções em uma rede para concluir suas tarefas.
- **Escalabilidade**: Como a computação na nuvem é distribuída, os usuários podem usar a capacidade que necessitam, sob demanda.
- **Disponibilidade**: A computação em nuvem tornou os recursos de computação e os dados muito mais disponíveis.
- **Resiliência**: Computadores tradicionais estão sujeitos a falhas, pois a falha de um único componente de hardware pode tornar o computador inteiro inutilizável. A computação distrbuída elimina esse problema modularizando o poder de processamento e criando armazenamentos de dados redundantes para em caso de falha, não perder acesso aos dados.

## Benefícios da computação em nuvem para usuários
- **Conveniência**: Os usuários podem acessar seus dados e aplicativos de qualquer lugar com uma conexão de Internet e em qualquer dispositivo. Isso elimina a necessidade de carregar dispositivos de armazenamento físico ou instalar software em vários dispositivos. 
- **Economia de custo**: Os usuários não precisam investir em hardwares e softwares caros. Eles podem pagar apenas pelos recursos que usam, resultando em economia de custo significativa.
- **Flexibilidade**: Os usuários podem acessar recursos de computação de qualquer lugar com uma conexão de Internet e em qualquer dispositivo. Essa flexibilidade permite trabalho remoto e permite que eles colaborem facilmente com outras pessoas.
- **Confiabilidade**: Os usuários podem esperar que a nuvem esteja sempre disponível e ofereça resiliência contra falhas. Isso ajuda a assegurar que eles sempre possam acessar seus recursos computacionais, eliminando o risco de tempo de inatividade, que pode impactar a produtividade e atrapalhar as atividades pessoais.
- **Segurança**: Os dados e informações pessoais dos usuários são protegidos pelas robustas medidas de segurança que os provedores de serviço de nuvem oferecem.

## Benefícios da computação em nuvem para empresas
- **Inovação**: As empresas podem experimentar novas tecnologias e serviços de forma rápida e fácil, o que as ajuda a inovar e responder às tendências de mercado com mais rapidez.
- **Agilidade**: As empresas podem permanecer ágeis, responsivas e competitivas. Como os provedores de nuvem oferecem uma ampla gama de serviços, as empresas podem se adaptar e crescer para que seus negócios cresçam e mudem conforme as demandas do mercado.
- **Colaboração**: Os funcionários podem acessar recursos computacionais de qualquer lugar com uma conexão de Internet, possibilitando trabalho remoto e colaboração com parceiros e clientes.

## Como a computação em nuvem beneficia usuários e empresas
- **Conexão única**: Os usuários configuram uma conta única com vários provedores de nuvem para obter acesso a uma variedade de softwares e serviços. Os usuários só precisam de uma única conta do usuário em todos os seus dispositivos. Os usuários podem até efetuar login em seus dispositivos usando sua conta em nuvem. 
- **Armazenamento em nuvem**: Uma vez conectados, os usuários armazenam documentos, imagens, dados e até mesmo transmitem mídia por meio do armazenamento em nuvem. As empresas podem usar armazenamento em nuvem seguro para ativos corporativos como documentos e mídias. 
- **Aplicativos**: Muitos provedores de nuvem oferecem softwares pela nuvem, como aplicativos de processamento de texto e planilhas. As empresas publicam seus aplicativos corporativos internos na nuvem e os entregam aos funcionários no mundo todo. 
- **Cargas de trabalho**: Uma carga de trabalho é uma tarefa específica que um sistema de computador executa. Os provedores de nuvem oferecem cargas de trabalho como backups automáticos e armazenamento de imagens para dispositivos móveis que podem salvar os usuários de catástrofes. As empresas desenvolvem microsserviços, pequenos pacotes de software que fazem algo específico, e os publicam na nuvem. Os usuários podem aumentar e diminuir a capacidade dessas cargas de trabalho com base na demanda e na necessidade. 
- **Dispositivos conectados**: Muitos provedores de nuvem permitem conectividade de dispositivo para que um dispositivo possa enviar dados e se comunicar com outro dispositivo. Ao usar a nuvem, os dispositivos dos usuários podem funcionar como extensões uns dos outros com conectividade perfeita.
- **Desempenho**: Como a nuvem é distribuída e utiliza a capacidade de milhares de computadores, tanto usuários quanto empresas conseguem obter aplicativos e serviços com excelente desempenho. O armazenamento em nuvem é replicado em todo o mundo, então os downloads podem levar minutos ou segundos, em vez de horas. As rede de nuvem estão disponíveis em muitos locais, então a latência de rede, ou atrasos, são mínimos.
